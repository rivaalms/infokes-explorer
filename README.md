# infokes

## Install dependencies

```bash
bun install
```

## Running Migrations

```bash
bun run server:db:migrate
```

## Seeding Data

```bash
bun run server:db:seed
```

## Run the app

### Front-end App

```bash
bun run app:dev
```

### Back-end Server

```bash
bun run server:dev
```

## Tech Stacks & Packages

### Front-end App
